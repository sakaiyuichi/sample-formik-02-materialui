# sample-formik-02-materialui
formikのサンプル（MaterialUI利用）

## インストール

```
> yarn add formik
> yarn add yup
> yarn add @material-ui/core

以下は datepicker を利用するために必要
> yarn add @date-io/date-fns@1.x
> yarn add date-fns
> yarn add @material-ui/pickers
> yarn add @material-ui/icons
```
